Mi Proyecto Web

Primer desafío práctico con Git, GitHub y buenas prácticas de codificación.

Propósito del proyecto

Este sitio web básico permite aplicar una estructura HTML ordenada, documentada y versionada mediante Git.

Buenas prácticas aplicadas

  • Uso correcto de la estructura HTML5.
  • Indentación consistente para mejorar la legibilidad.
  • Separación clara entre encabezado, contenido principal y pie de página.
  • Documentación del proyecto mediante archivo README.md.

Control de versiones

El proyecto utiliza Git para registrar cambios mediante commits y GitHub para alojar el repositorio remoto.

Desarrollado para la asignatura Aplicaciones y Tecnologías de la Web.
