# Mi Proyecto Web

## Descripción

Este repositorio corresponde al primer taller formativo de la asignatura Aplicaciones y Tecnologías de la Web. Su objetivo es aplicar fundamentos de Git y GitHub mediante la creación de un proyecto web básico, el uso de commits, ramas, fusión de cambios y documentación técnica.
